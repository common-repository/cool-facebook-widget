Facebook Like Box hidding from the beginning and it will be moving in smoothly when user reach bottom of the page.

== Description ==

* Facebook Like Box hidding from the beginning and it will be moving in smoothly when user reach bottom of the page.
* You can modify the box styling and set fanpage URL in Admin > Settings > Cool Facebook Settings
* To place facebook box fixed in sidebar, use the box widget in Appearance > Widgets

== Installation ==

1. Upload folder `cool-facebook-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
